PCET documentation: PCETupdate

PCETupdate

PCETsys = PCETupdate(PCETsys, 'var1', data1,..., 'varN', dataN)
returns the system PCETsys with updated/modified uncertainties of the 
initial conditions or parameter values as specified by the input 
arguments. The input values data1 to dataN have to match the distribution
of the corresponding variable in PCETsys.

NOTE: PCETupdate cannot change the system's dynamics, the order of the 
      polynomial chaos expansion, or the disbribution of a variable.
      Use PCETcompose instead!
