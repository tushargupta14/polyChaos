PCET documentation: writeMCRHSfile

writeMCRHSfile

writeMCRHSfile(PCETsys, 'filename') 
writes a function-file filename.m containing the states' scalar
right-hand-sides of the system defined in PCETsys. The file will be
created in the current directory. This function-file is required for a
simulation using PCETsimCollocation or PCETsimMonteCarlo.
